# Roblox Legends Archive
Un projet documentant les mythes et légendes de l'ancien Roblox.

## Installation
Instructions pour installer et configurer le projet :
1. Cloner le dépôt : `git clone https://github.com/zdn-chb/Projet-2-NSI.git`
2. Aller dans le dossier du projet : `cd Projet-2-NSI`

## Utilisation
Les utilisateurs peuvent :
- Explorer les histoires sur les figures légendaires de Roblox.
- Lire leurs récits et théories.
- Participer aux discussions sur les hackers et mythes célèbres.

## Fonctionnalités
- Base de données interactive des légendes de Roblox.
- Système de recherche des personnages emblématiques.
- Forum communautaire pour partager et échanger des théories.
- Interface dynamique en HTML, CSS et JavaScript.

## Contribution
Si vous souhaitez contribuer, voici les étapes à suivre :
1. Forkez le dépôt.
2. Créez une nouvelle branche : `git checkout -b nouvelle-fonctionnalite`
3. Apportez vos modifications et validez-les : `git commit -m "Ajout d'une nouvelle légende"`
4. Poussez vers GitHub : `git push origin nouvelle-fonctionnalite`
5. Soumettez une demande de fusion (pull request).
